About This Website

This website was created as a project to practice web development skills, including HTML structure, CSS styling, and navigation design.

It focuses on Apple’s AirPods lineup, highlighting the different models released over the years and their key features.

The site demonstrates requirements such as tables, lists, styled divs, spans, images, and external CSS.
